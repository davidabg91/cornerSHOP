# THE CORNER - Grocery Store Sofia

Modern, fast, and responsive informational website for "The Corner" grocery store located in Sofia (NPZ Iskar).

## Features
- **Modern UI**: Navy blue & fresh green corporate identity.
- **Responsive Design**: Optimized for mobile and desktop.
- **Dynamic Content**: Sections for products, business focus, and upcoming features.
- **Location Integration**: Direct Google Maps navigation and interactive embed.
- **Interactive Forms**: Product suggestion and notification systems.

## Local Development
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
